How can I help?

You can contribute in multiple ways:

  • First, by playing! The more people we have playing, the easier it is for anyone to find a match or get help.
  • If you are a programmer (or would like to be one), you can see GEMP's code base on GitHub here. Pull requests are welcome. Coordinate with Ketura on the PC Discord if you are looking for specific tasks to perform.
  • If you're interested in playtesting, go to My Account -> Playtesting and opt-in. This will give you access to three PLAYTEST formats, one each for PLAYTEST - Fellowship Block (PC), PLAYTEST - Movie Block (PC), and PLAYTEST - Expanded (PC).

    These formats have the latest untested errata and upcoming Virtual Sets from the Player's Council, and we can always use more people testing things out.
  • If you have ideas on cards that you think should be errata'd, or if you have Opinions on the PC's big gameplay initiatives, then come let us know on the PC Discord. We are always happy to hear people's proposals and hear your thoughts on our own ideas.
  • If you would like to join the PC on a more involved level, go fill out the form linked on this page.
